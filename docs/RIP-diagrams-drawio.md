## Диаграмма 2 — Сценарий работы
### Слайд 2 "Как это работает"

**Что рисовать:** три прямоугольника в ряд слева направо, между ними стрелки.

**Блок 1 (левый):**
- Прямоугольник с закруглёнными углами, синяя заливка
- Заголовок крупно: "1. Новый проект"
- Текст мелко: "Менеджер вводит компанию заказчика"

**Стрелка 1→2:**
- Горизонтальная стрелка вправо
- Подпись под стрелкой: "система ищет в графе"

**Блок 2 (центр):**
- Прямоугольник с закруглёнными углами, тёмно-синяя / индиго заливка
- Заголовок крупно: "2. Граф находит путь"
- Текст мелко: "Иванова знакома с Петровым, Петров — партнёр Сидорова"

**Стрелка 2→3:**
- Горизонтальная стрелка вправо
- Подпись под стрелкой: "генерируется отчёт"

**Блок 3 (правый):**
- Прямоугольник с закруглёнными углами, янтарная / тёмно-зелёная заливка
- Заголовок крупно: "3. Отчёт готов"
- Текст мелко: "Точка входа найдена. PDF с картой связей"

**Подпись под всей схемой:**
- По центру, курсив, серый: "от сигнала до точки входа — минуты, не дни"

**Размеры:** все три блока одинаковой ширины и высоты. Стрелки той же длины.

## Диаграмма 4 — Безопасность данных
### Слайд 4 "Pseudonymization"

**Что рисовать:** три зоны в ряд слева направо.

**Зона 1 (левый блок):**
- Прямоугольник с зелёной пунктирной обводкой, тёмная заливка
- Заголовок сверху: "🔒 Ваш сервер"
- Внутри список (выровнен по левому краю):
  - Иванов Иван Иванович
  - +7 (999) 123-45-67
  - ИНН 7730588444
  - ivanov@company.ru

**Стрелка вправо (шифрование):**
- Толстая стрелка вправо →
- Над стрелкой: "замена на коды"
- Под стрелкой: иконка замка 🔐 (или нарисовать замок)

**Центральный блок:**
- Прямоугольник, тёмно-серая заливка
- Заголовок сверху: "🤖 AI видит только"
- Внутри список:
  - PERSON_001
  - PHONE_001
  - INN_001
  - EMAIL_001
- Мелкий текст снизу внутри блока: "реальные данные недоступны"

**Стрелка влево (расшифровка):**
- Толстая стрелка влево ←  (в правую сторону от центра)
- Над стрелкой: "обратная замена"

**Зона 3 (правый блок):**
- Такой же как зона 1 — зелёная пунктирная обводка
- Заголовок сверху: "✅ Готовый отчёт"
- Внутри текст:
  - Иванов Иван Иванович
  - связан с Петровым через...
  - точка входа найдена

**Подпись под всей схемой:**
- Жирный белый текст по центру: "Имена, телефоны, ИНН — никогда не покидают ваш сервер"

**Важно:** все три блока одинаковой высоты. Центральный блок чуть уже крайних.

## Общие советы по draw.io

**Тёмный фон:**
Правой кнопкой на холсте → Edit Background → выбрать тёмный цвет (#1a1a2e или #0d0d1a)

**Пунктирная линия:**
Выделить линию → Format → Connection → поставить Dashed

**Бейдж (маленький тег):**
Нарисовать маленький прямоугольник с закруглёнными углами, разместить поверх основного блока в углу

**Одинаковый размер блоков:**
Выделить все блоки → правая кнопка → Edit Style → вручную задать одинаковый width и height

**Экспорт:**
File → Export As → PNG, галочка "Transparent Background" если фон слайда тёмный
